CV Batman

CV Batman
Este sitio web lo realizamos en la mesa de trabajo n°22 y consistió en refactorizar una página de currículum vitae utilizando el preprocesador Sass. A partir del trabajo “CV-Batman” realizado en clases anteriores, se reescribieron los estilos aplicando variables, funciones y anidamiento para optimizar la estructura del código y mejorar su mantenimiento. El objetivo de la actividad fue incorporar el uso de Sass como herramienta que permite generar hojas de estilo más organizadas, reutilizables y fáciles de actualizar, manteniendo el mismo resultado visual que el diseño original.
